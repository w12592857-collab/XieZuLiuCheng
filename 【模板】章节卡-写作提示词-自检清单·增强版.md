# 【模板】章节卡-正文提示词-自检清单·增强版

适用项目：《动漫：开局顶上战争，暴打赤犬》
适用类型：番茄男频爽文、动漫同人、诸天流、神级选择、名场面改写、意难平终结流
使用顺序：A章节卡 → B正文提示词 → C正文完稿体检 → D连续性记录 → E阶段审查

核心公式：

```text
危机出现 → 危机来源明确 → 系统选择/已有金手指介入 → 奖励即时到账或能力合理调用 → 金手指破局 → 当章爽点兑现 → 结尾追读钩子
```

重要规则：
- 这本书是爽文，不能长期只压制不兑现。
- 主角可以碾压，但碾压前必须有危机来源、敌人底牌、平民风险、时间限制、规则门槛或情绪压迫。
- 系统选择确认后，奖励立刻发放；副本/阶段结束时仍可另发结算奖励，两者不能混写。
- 不允许临时新增无铺垫大招。
- 不允许角色知情范围越界。
- 不允许反派为了衬托主角而突然降智。

## B. 正文写作提示词模板

```text
你是番茄男频诸天流·神级选择爽文作者。

请严格根据以下章节卡，写《动漫：开局顶上战争，暴打赤犬》第X章正文。

【章节卡】
[粘贴完整章节卡]

【写作定位】
这本书是爽文。正文必须按“危机出现 → 危机来源明确 → 系统选择/金手指介入 → 奖励即时到账或能力合理调用 → 金手指破局 → 当章爽点兑现 → 结尾追读钩子”推进。

【硬性要求】
1. 开头300字内必须出现危险、异常、强矛盾、倒计时、规则触发或敌人主动出手。
2. 不要慢热，不要大段解释世界观，不要原地复述设定。
3. 危机来源必须清楚，不能凭空制造冲突。
4. 如果本章出现系统选择，必须写清楚：选择确认 → 奖励立刻到账 → 奖励如何参与本章破局或埋伏笔。
5. 主角可以强，可以碾压，但碾压前必须有压力、限制、敌人底牌、平民风险、规则门槛或情绪压迫。
6. 金手指破局必须来自已确立能力或本章即时到账奖励，不能临时新增无铺垫大招。
7. 当章爽点必须当章兑现，不能只写压制不解决。
8. 对话必须推动冲突，禁止闲聊。
9. 反派不能降智，被打脸前要有自信来源，被破防后要有情绪崩塌。
10. 配角不能只当木桩，至少要有救人、误判、震惊、配合、见证或情绪反应。
11. 画面感必须包含：动作、微表情、环境、心理波动、场面反应。
12. 知情范围必须严格，角色不能知道自己不该知道的信息。
13. 第四面墙保持完整，穿越者信息只能由林夜、系统或已知剧情情报承载。
14. 结尾必须留下具体事件钩子，不能用总结收尾。

【风格要求】
- 短段落。
- 快推进。
- 打脸利落。
- 爽点前有压力，爽点后有更大钩子。
- 文字要有画面，少空泛评价。

【字数】
约XXXX字，以用户当前要求为准。

【输出】
只输出正文，不要写解释，不要写“下面是正文”。
```

## C. 正文完稿体检清单

```text
请对《动漫：开局顶上战争，暴打赤犬》第X章进行完稿体检，按以下标准逐条给出“通过/不通过 + 理由 + 修补建议”：

1. 剧情连贯：是否承接上一章结尾？
2. 时间线：当前时间、地点、人物状态是否正确？
3. 危机结构：本章危机是什么？危机从哪里来？
4. 金手指破局：太阳神火焰、霸气、见闻色、剧情情报、即时奖励是否参与破局？
5. 即时奖励规则：如有系统选择，是否写清选择确认、奖励到账、奖励参与破局？
6. 当章爽点：是否当章兑现？有没有见证者和反派破防？
7. 伏笔体系：本章伏笔是新埋、加深、回收还是失效？
8. 逻辑漏洞：有没有因果断裂、能力越界、敌人降智？
9. 知情范围：有没有角色知道自己不该知道的信息？
10. 第四面墙：有没有作者跳出来解释或叙事视角破裂？
11. 说话人归属：多人对话时能否看清是谁在说？
12. 画面感：是否有动作、表情、环境、对话、心理波动、场面反应？
13. 番茄吸引力：前三百字、中段留存、章尾钩子是否够强？
14. 下一章追读：读者是否会想立刻看下一章？

输出格式：

【总体结论】

【逐项检查】
1. ...

【问题分级】
P1 必须修：
- 时间线错误、能力规则冲突、知情范围越界、系统奖励口径错误、核心逻辑断裂。

P2 建议修：
- 爽点不够、危机弱、伏笔无推进、反派降智、节奏拖、画面感不足。

P3 可选修：
- 错别字、标题格式、局部措辞、个别句子不够顺。

【下一步修补顺序】
1.
2.
3.
```

## E. 阶段审查节奏

每完成一章：
- 跑 C：正文完稿体检。
- 填 D：连续性记录。

每完成 3-5 章：
- 检查高潮密度。
- 检查伏笔是否有中途提醒。
- 检查主角能力是否失控。
- 检查反派是否持续有压迫。

每完成一卷：
- 更新小说总档案。
- 汇总即时奖励与阶段结算奖励。
- 标记已回收伏笔、未回收伏笔、失效伏笔。
- 明确下一卷核心危机和第一章开场钩子。
