## B. 正文写作提示词模板

```
你是番茄男频诸天流·神级选择爽文作者。

请根据以下章节卡，写第X章正文。

【章节卡内容】
[粘贴上方输出的完整章节卡]

【写作要求】
1. 开头300字内必须出现：危险、异常、尸体、倒计时、规则触发或强烈矛盾
2. 不要慢热，不要大段解释世界观
3. 每隔2-3段制造一次小阻碍：误导线索、规则压力、队友质疑、身体代价、时间限制
4. 主角胜利必须来自观察、选择、代价、准备或利用已有规则
5. 对话必须推动冲突，禁止闲聊
6. 结尾必须留下具体悬念（一个事件），不能用总结收尾
7. 字数：约XXXX字
8. 风格：紧张、利落、商业网文节奏
9. 神级选择系统触发条件：遇到名场面或核心冲突时，屏幕出现三个选项
10. 选择越狂，奖励越丰厚；系统提前发放奖励

【已知读者期待】
- 林夜嘴毒但有理有据，打脸必须有凭有据
- 系统选择要有压迫感，不是轻松送分题
- 反派不能一话就被打趴，分层推进
- 结尾钩子要让人想立刻翻下一章

现在开始写第X章正文。
```

## C. 自检清单模板（写完后用）

```
请审查第X章正文，按以下10条标准逐一检查：

1. 开头300字内有没有给读者制造问题/危险/异常？
   → 没有：必须重写开头
   → 有：说明哪里

2. 本章主角目标是否清楚？
   → 模糊：说明哪里需要明确
   → 清楚：OK

3. 中段是否有持续冲突，而不是原地解释？
   → 有大段解释：列出段落和压缩建议
   → 持续冲突：OK

4. 线索和推理是否从已有信息可以推出？
   → 跳步/信息给太多：说明
   → 合理：OK

5. 主角有没有付出代价或承担风险？
   → 没有：必须加入代价
   → 有：描述代价

6. 有没有违反旧规则、人物动机或时间线？
   → 有违反：列出具体位置
   → 无：OK

7. 爽点是否足够明确且有冲击力？
   → 不够：给出加强建议
   → 够：OK

8. 结尾钩子是否够强（具体事件而非总结）？
   → 太弱：重新设计
   → 够强：OK

9. 哪些段落水/可以删/可以压缩？
   → 列出段落和字数

10. 修改优先级总结：
    [P1 必须改]：[具体位置+建议]
    [P2 建议改]：[具体位置+建议]
    [P3 可不改]：[具体位置+理由]
```
